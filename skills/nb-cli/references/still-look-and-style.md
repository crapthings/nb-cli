# Still Look And Style Language

Use this file when a still image needs finish, grading, optical, or single-frame style vocabulary.

Ignore sound, editing transitions, story devices, and camera moves. Keep the language tied to a single final image.

## Optical and finish cues
- `film grain`: subtle analog texture.
- `bokeh`: pleasing out-of-focus highlights.
- `lens flare` / `anamorphic flares`: flare streaks or glow from bright sources.
- `bloom` / `glow`: soft bleed around highlights.
- `vignette`: darker or softer frame edges.
- `motion blur`: use only if the still image should imply movement.
- `chromatic aberration`: slight color fringing for stylized or vintage looks.
- `lens distortion`: wide-angle stretch or edge warp.
- `HDR look`: detail-rich highlights and shadows; use carefully.
- `reflections`: mirrored surfaces in water, glass, chrome, or wet ground.
- `refractions`: bent light through glass, liquid, or crystal.

## Color and grade language
- `color correction`: neutral, technically clean baseline.
- `color grading`: creative tonal treatment.
- `desaturation`: muted color intensity.
- `sepia tone`: nostalgic warm-brown aging.
- `vintage film look`: analog wear, grain, fade, and light leaks.
- `amber phosphor CRT`: monochrome amber monitor glow, scanlines, curved glass, phosphor ghosting.

## Single-frame style families
- `film noir`: hard contrast, shadows, noir atmosphere.
- `neo-noir`: contemporary noir with darkness, reflections, neon, and moody ambiguity.
- `German Expressionism`: distorted, shadow-heavy dramatic stylization.
- `surrealism`: dreamlike, uncanny, illogical imagery.
- `cinema verite`: raw, minimally polished realism.
- `Italian neorealism`: grounded, lived-in realism with modest styling.
- `giallo`: vivid color and stylized thriller imagery.
- `minimalism`: sparse restrained visual language.
- `maximalism`: dense detailed visual abundance.
- `arthouse`: image-first stylized treatment.

## Prompting guidance
- Use style terms to refine a strong base prompt, not replace subject and composition.
- Pair broad style labels with concrete visual traits, such as `neo-noir, wet reflections, low-key lighting`.
- Avoid heavy grading language for accurate product imagery unless the user asks for it.
