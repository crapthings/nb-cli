# Supported Aspect Ratios for nb-cli

Ordered by the leading dimension (width) from largest to smallest.

### Horizontal
- `21:9`: Ultra-wide cinematic
- `16:9`: Standard widescreen
- `8:1`: Panorama
- `5:4`: Classic print
- `4:3`: Retro monitor
- `4:1`: Wide panorama
- `3:2`: 35mm film

### Square
- `1:1`: Social media post / Profile picture

### Vertical
- `9:16`: Story / Reel / TikTok
- `4:5`: Instagram portrait
- `3:4`: Retro portrait
- `2:3`: Classic portrait
- `1:4`: Tall panorama
- `1:8`: Ultra-tall panorama
