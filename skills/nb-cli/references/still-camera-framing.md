# Still Camera And Framing Language

Use this file when a prompt needs single-image camera-angle or framing language.

Ignore motion-dependent terms such as dolly, tracking, pan, tilt, crane, whip pan, handheld movement, or long-take language.

## Angles and vantage points
- `eye-level`: neutral, natural perspective.
- `high-angle`: look down on the subject; smaller or more vulnerable feel.
- `low-angle`: look up at the subject; larger or more dominant feel.
- `worm's-eye view`: ground-level extreme low angle for dramatic scale.
- `bird's-eye view`: straight-down abstract overview.
- `overhead`: top-down layout view for products, food, desks, maps, and arranged scenes.
- `aerial`: elevated wide establishing-like still for geography, cities, or landscapes.
- `head-on`: front-facing symmetrical viewpoint.
- `profile`: side-view silhouette or side portrait.

## Shot size and framing
- `wide shot` / `extreme long shot`: environment dominates.
- `long shot` / `full shot`: full body or full object with context.
- `american shot` / `cowboy shot`: mid-thigh up; useful for stance and gesture.
- `medium shot`: waist up.
- `medium close-up`: chest up.
- `close-up`: face or object detail fills the frame.
- `extreme close-up`: isolate a detail such as eyes, hands, texture, or a product feature.
- `head shot`: shoulders up portrait.
- `insert shot`: detail-focused object close-up.
- `two-shot` / `three-shot`: show multiple subjects in one still composition.

## Static framing language
- `centered composition`: formal and deliberate.
- `symmetrical framing`: balanced and graphic.
- `off-center framing`: more editorial or tension-building.
- `top-down flat lay`: overhead arrangement on a surface.
- `usable negative space`: leave room for UI or text overlay.

## Prompting guidance
- Combine one angle term and one framing term instead of stacking many synonyms.
- Use camera language only if it improves the requested image materially.
- For product, icon, and UI work, prefer simple stable terms like `centered`, `three-quarter angle`, `top-down`, or `close-up detail`.
